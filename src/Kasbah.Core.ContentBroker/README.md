# Kasbah.Core.ContentBroker

This module provides a single interface to access all internal services.

It will also handle caching of content including cache invalidation and upating.

## Dependencies

 * `Kasbah.Core`
 * `Kasbah.Core.Events`
 * `Kasbah.Core.Tree`
 * `Kasbah.Core.Index`
 * `Kasbah.Core.Cache`
